CodysTools Test Page

Use this page to test your CodysTools extension and mods

How to Test CodysTools

Install the CodysTools extension in Chrome
Click the CodysTools icon in your browser toolbar
Enable the "Example Mod" in the popup
Refresh this page to see the mod in action
You should see a banner appear in the top-right corner

Extension Features

  • Modular architecture with GitHub integration
  • Automatic update checking and installation
  • Site-specific mod targeting
  • CSS and JavaScript injection
  • Configurable settings per mod
  • Import/export functionality
  • Conflict detection
  • Modern, responsive UI

Testing the Example Mod

The example mod demonstrates basic CodysTools functionality:

  • CSS Injection: Adds custom styling for the banner
  • JavaScript Execution: Creates and manages the banner element
  • Settings Support: Configurable text, theme, and animation speed
  • Event Handling: Click to close functionality
  • Auto-cleanup: Banner disappears after 5 seconds

Installing Custom Mods

To install mods from GitHub repositories:

Click the "+" button in the CodysTools popup
Enter a repository in format: username/repository
Optionally specify a branch (defaults to "main")
Click "Install Mod"

The mod must have a mod.json file in the repository root.

Mod Development

Create your own mods by:

Creating a GitHub repository
Adding a mod.json configuration file
Including CSS and/or JavaScript files
Testing with CodysTools

Settings and Configuration

Access advanced settings by:

Clicking the gear icon in the popup
Or right-clicking the extension icon and selecting "Options"

Configure update intervals, performance settings, and more.
